# Романтический сайт "Прости меня" 💕

Интерактивный сайт для извинений с анимированными эффектами.

## Как загрузить на GitHub Pages:

### Шаг 1: Создайте репозиторий на GitHub
1. Зайдите на github.com и войдите в свой аккаунт
2. Нажмите "New repository" (зеленая кнопка)
3. Назовите репозиторий, например: `romantic-site`
4. Выберите "Public"
5. Нажмите "Create repository"

### Шаг 2: Загрузите файлы
1. В созданном репозитории нажмите "uploading an existing file"
2. Перетащите все файлы из папки romantic-site:
   - index.html
   - success.html
   - background.jpg
   - photo1.png
   - photo2.png
3. Нажмите "Commit changes"

### Шаг 3: Включите GitHub Pages
1. В репозитории перейдите в "Settings" (вкладка сверху)
2. В левом меню найдите "Pages"
3. В разделе "Source" выберите "main" или "master" ветку
4. Нажмите "Save"
5. Подождите 1-2 минуты

### Шаг 4: Получите ссылку
Ваш сайт будет доступен по адресу:
```
https://ваш-username.github.io/romantic-site/
```

## Функционал сайта:

### Главная страница (index.html):
- ✨ Фоновое фото с затемнением
- ❤️ Летающие сердечки (40+ штук)
- 💚 Зеленая кнопка "Хорошо прощаю" - ведет на страницу успеха
- ❤️ Красная кнопка "Жок, сам виноват":
  - При нажатии перемещается в случайное место
  - Меняет текст рандомно: "Жооок", "Кешырмимм", "Все ровно жок", "Неет"
  - С каждым нажатием зеленая кнопка увеличивается в 1.2 раза

### Страница успеха (success.html):
- 💖 Ярко-розовый градиентный фон
- 🖼️ Анимированные фотографии (меняются каждые 0.5 сек как гифка)
- 💕 Романтическая надпись с эффектом свечения
- ✨ Плавающие и фоновые сердечки

## Технологии:
- HTML5
- CSS3 (анимации, градиенты, эффекты)
- Vanilla JavaScript (без библиотек)
